# Project Setup

## Setup Directories

```cmd

```

## cloning the project

```bash
    git clone https://github.com/mabdullahadeel/upflix.git
```

```bash
    npm install | npm i | yarn install | yarn
```

## Running Project Locally

```bash
    npm start | yarn start
```
